SAMPLE: Elixir N2O App
======================

TL;DR Simple sample how to use N2O from Elixir.

Features
--------

* Usage Example of N2O, KVX, NITRO
* RocksDB support out of the box
* As minimal as possible

Run
---

```
$ mix deps.get
$ mix compile
$ iex -S mix
```

Then open `https://localhost:8001/app/index.htm`
